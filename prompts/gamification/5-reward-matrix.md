# 🎯 Gamification Designer — Step 5: Create Unified Reward Matrix

Combine all prior data: actions, quests, and achievements.

## 🧩 Instructions

Each row shows how XP, coins, and achievements are tied together.

## 📤 Output Format

Output as **CSV**:

```
ActionOrQuest,EventTrigger,Type,XP,Coins,RelatedAchievement,Notes
```

Example:

```
Complete Lesson,lesson_completed,action,50,10,Lesson Master,Core loop
Weekly Challenge,weekly_activity,quest,200,100,Consistency Star,Encourage habit
30-Day Streak,streak_30days,achievement,500,150,Endurance Badge,Major milestone
```
