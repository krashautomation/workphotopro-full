## 🧭 Instructions

List all distinct user actions or feature interactions, and estimate their:

- Frequency (daily, weekly, occasional)
- Effort (low, medium, high)
- Value to User (1–5)
- Value to App (1–5)
- Suggested event trigger name (for analytics/events)

## 📤 Output Format

Output as **CSV** with these headers:

```
Action,Frequency,Effort,ValueToUser,ValueToApp,EventTrigger
```

Example:

```
Complete Lesson,Daily,Medium,5,5,lesson_completed
Review Vocabulary,Daily,Low,4,4,vocab_reviewed
Invite Friend,Occasional,High,4,5,friend_invited
```
